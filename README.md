# calculadora-de-prazos
Funcionalidades: - Calcula vencimento a partir da data da intimação - Mostra quantos dias restam para o prazo vencer - Considera feriados nacionais e estaduais por tribunal - Aplica suspensão do recesso forense automaticamente - Linha do tempo visual, dia a dia, com marcação de feriados e recesso - Busca por tipo de prazo
